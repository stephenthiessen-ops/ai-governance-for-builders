# Human-in-the-Loop Framework

AI must not operate autonomously in high-impact domains.

---

## Review Requirements by Tier

Tier 1:
- Optional review

Tier 2:
- Sample-based validation
- Weekly review of outputs

Tier 3:
- Mandatory approval before action
- Documented decision log

---

## Escalation

If output confidence < threshold:
- Flag for manual review
- Log override decision
