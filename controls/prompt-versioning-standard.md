# Prompt Versioning Standard

All production prompts must:

- Be stored in version control
- Have semantic versioning (v1.0.0)
- Include change notes
- Include risk classification

---

## Required Metadata

- Owner
- Risk Tier
- Intended Use
- Model Version
- Last Review Date

Prompt changes must trigger review if:
- Risk tier increases
- External exposure changes
- Data scope expands
