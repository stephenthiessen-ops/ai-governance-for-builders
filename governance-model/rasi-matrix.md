# AI Governance RASI Matrix

Defines accountability boundaries to prevent governance ambiguity.

| Activity | AI Platform | Engineering | Security | Legal | Exec Sponsor |
|-----------|------------|------------|----------|-------|--------------|
| Risk Tier Assignment | R | A | C | C | I |
| Model Approval | R | C | C | A | I |
| Prompt Updates | R | A | C | I | I |
| External Exposure Approval | C | C | R | A | I |
| Audit Review | C | C | R | A | I |
| Incident Response | C | R | A | C | I |
| Decommissioning | R | A | C | I | I |

R = Responsible  
A = Accountable  
C = Consulted  
I = Informed  

Governance fails when accountability is ambiguous.
